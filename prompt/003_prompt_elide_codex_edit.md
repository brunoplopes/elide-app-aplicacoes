## No tipo de Usuário

### Estabelecimento

Funcionalidades:

- Cadastro
- Envio de documentação
- Aprovação pelo administrador
- Gestão de produtos
- Gestão de categorias
- Gestão de complementos
- Gestão de horários
- Controle de estoque
- Promoções
- Recebimento de pedidos
- Alteração de status do pedido
- Financeiro
- Relatórios
- Dashboard
- Avaliações

Backend:
- Crie uma nova RoleName STORE_USER no backend para controle de permissão
- implemente todas as funcionalidades do Estabelecimento no backend

Frontend
- Implemente as chamadas para api do Estabelecimento no frontend
