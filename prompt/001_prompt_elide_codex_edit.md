### Design de todas as página do projeto frontend deve respeitar as regras abaixo:

- copia design da pagina https://elide-fast-delivery.lovable.app/
- copia estilo dos fomularios da pagina https://elide-fast-delivery.lovable.app/login
- copia estilo do header da pagina https://elide-fast-delivery.lovable.app/
- copia estilo do footer da pagina https://elide-fast-delivery.lovable.app/ e apontar para rotas corretas
- as paginas devem ser responsivas
- Componentes reutilizáveis
- Utilizar arquitetura de projeto
- dentro de pages criar componentes onde cada pagina tem sua pasta e dentro da pasta separar os arquivos
  - component.ts
  - template HTML
  - estilo CSS
